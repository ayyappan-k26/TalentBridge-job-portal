# TalentBridge - Online Job Portal with Intelligent Matching

TalentBridge is a full-stack web-based job portal that connects job seekers with employers through an intelligent job matching system. Built as a final year MCA project, it streamlines the recruitment process for both students and companies.

## 🚀 Features

- **Student/Job Seeker Module**
  - User registration & login
  - Profile creation with resume upload
  - Browse and search job listings
  - Apply to jobs directly through the platform
  - Track application status

- **Company/Employer Module**
  - Company registration & login
  - Post new job openings
  - View applicant profiles and resumes
  - Manage posted jobs

- **Admin Panel**
  - Manage users (students & companies)
  - Monitor job postings and applications
  - Platform oversight and control

- **Intelligent Matching**
  - Matches job seekers to relevant job postings based on skills/profile

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL / SQLite
- **Other:** Jinja2 Templates

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/ayyappan-k26/TalentBridge-job-portal.git

# Navigate to project folder
cd TalentBridge-job-portal/backend

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
